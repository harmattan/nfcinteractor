Detailed Description

Parse the contents of an NDEF message and return the contents of known records in human-readable textual form.

The output contains information about all the records contained in the message. More details are reported for records of type: Uri, Text, Smart Poster, Image and Mime/vCard.

The NfcTargetAnalyzer has a similar task, but returns the general tag information in textual form.
